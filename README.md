# 🌸 Floria - Digital Flower Bouquet

Floria adalah website hadiah digital bertema buket bunga yang dibuat menggunakan **HTML, CSS, dan JavaScript**. Website ini menghadirkan animasi yang halus, tampilan yang estetik, serta pengalaman interaktif yang cocok untuk diberikan kepada orang tersayang atau dijadikan sebagai proyek portofolio Front-End.

## ✨ Fitur

* 🎁 Animasi pembukaan kotak kado.
* 💐 Buket bunga interaktif dengan efek animasi.
* 💌 Surat digital dengan efek mengetik.
* 🎵 Pemutar musik sederhana.
* 🎨 Pilihan tema warna buket.
* 🌙 Mode gelap (Dark Mode).
* 📸 Simpan tampilan buket sebagai gambar.
* 🍃 Efek kelopak bunga dan animasi yang ringan.

## 🛠️ Teknologi

* HTML5
* CSS3
* JavaScript
* html2canvas
* Font Awesome

## 📁 Struktur Folder

```text
bouquet project/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🚀 Cara Menjalankan

1. Clone repository ini.
2. Buka folder proyek.
3. Jalankan file **index.html** di browser.

Agar semua fitur berjalan dengan baik, disarankan menggunakan **Live Server** di Visual Studio Code.
